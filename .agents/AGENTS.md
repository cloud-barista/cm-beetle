# Antigravity Project Rules for CM-Beetle / Beetle UX Lab

## UI Typography & Font Size Rules (Beetle UX Lab)

1. **Tab Description Header Boxes**:
   - Single-line flex container (`flex flex-wrap items-center gap-x-3 gap-y-1.5 px-6 py-4.5 rounded-2xl`).
   - Title: `text-base font-extrabold text-text-main` accompanied by its tab Lucide icon (`w-5 h-5 text-emerald-500`).
   - Subtitle description: `text-sm text-text-muted`.

2. **Text Casing & Styling**:
   - Section headers & labels: Title Case only (e.g. `Spec`, `Root Disk`, `Security Group`, `Node Count`).
   - Project Terminology: Always use **`Nodes`** (or `Node Groups`) instead of `VM Instances` across all UI labels & tabs.
   - Avoid `ALL CAPS` or unnecessary `uppercase` on static labels.
   - Dynamic badges (status pills, CSP tags): `text-xs font-mono font-bold`.

3. **Card & Component Font Sizes**:
   - Main card titles: `text-sm font-extrabold text-text-main`.
   - Secondary badges & metadata pills: `text-xs font-mono font-bold` or `text-xs font-extrabold`.
   - Action buttons: `text-xs font-bold font-mono` with comfortable padding (`px-3 py-1.5` or `px-4 py-2`).

4. **Brand Palette**:
   - Emerald (#10b981) / Teal (#14b8a6) ONLY.
   - Never use `cyan-*`, `purple-*`, or `sky-*`.
